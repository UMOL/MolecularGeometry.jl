# uMOL-MolecularGeometry
A module for measuring the geometric properties of molecules.
The following quantities are supported:

+-----------------------------------+--------------------------------+
| Quantity                          | function                       |
+===================================+================================+
| geometric center                  |  measure(:geometric_center,obj)|
+-----------------------------------+--------------------------------+
